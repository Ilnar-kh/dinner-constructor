# Dinner Constructor — консольный генератор бизнес-ланчей

Java, Console Application, Collections

## О проекте

**Dinner Constructor** — учебный консольный проект, который представляет собой программу для генерации комбинаций бизнес-ланчей на основе доступных блюд.  
Приложение предназначено для автоматизации подбора меню и отработки базовых навыков Java-разработки.

## Назначение проекта

Проект разработан по техническому заданию и используется как учебный тренажёр для:
- коллекциями Java;
- разделением логики и пользовательского интерфейса;
- генерацией случайных данных.

## Сценарии использования

Приложение поддерживает два основных сценария:

### 1. Добавление блюд
Администрация ресторана добавляет доступные блюда:
- пользователь вводит тип блюда (например: Первое, Второе, Напиток);
- затем вводит название блюда;
- программа сохраняет блюдо в памяти приложения.

Тип и название блюда могут быть произвольными строками.

### 2. Генерация комбинаций бизнес-ланчей
После добавления блюд пользователь может сгенерировать комбинации:
- вводится количество комбинаций;
- затем вводится список типов блюд, которые должны входить в бизнес-ланч;
- каждый тип вводится с новой строки;
- для завершения ввода пользователь нажимает Enter на пустой строке.

Программа:
- случайным образом подбирает блюда каждого типа;
- формирует одну или несколько комбинаций;
- выводит результат в консоль.

Если пользователь вводит несуществующий тип блюда, программа выводит предупреждение и предлагает ввести другой тип.

## Консольный интерфейс

Приложение поддерживает следующие команды:
1 — добавить новое блюдо
2 — сгенерировать комбинации блюд
3 — выход из программы
